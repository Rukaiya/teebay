# README

## Backend
* Ruby version: 3.2.1

* System dependencies: rails-7.0.4, postgres

* Configuration
  * bundle install
  * copy example.database.yml as database.yml
  * provide username, password, host, port information

* Database creation:
  * rails db:create, 
  * rails db:migrate

* Database initialization:
  * rails db:seed

* Deployment instructions
  * rails s

## Frontend
* npm install
* npm run dev
